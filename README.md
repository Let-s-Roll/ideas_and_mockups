# Let's Roll - Ideas & Mockups

This repository serves as a centralized hub for interactive prototypes and design experiments for the **Let's Roll** ecosystem. It is hosted on GitHub Pages at [build.lets-roll.app](https://build.lets-roll.app/).

## Repository Structure

-   `index.html`: The **Product Roadmap** overview. This page serves as the entry point and navigation hub for all active mockups.
-   `event-mockup/`: Contains the **Event Discovery v1** prototype.
    -   `index.html`: The interactive mobile-viewport mockup for the event discovery feed.
-   `assets/`: Shared brand assets, logos, and local fonts.

## Features & Prototypes

### 1. Event Discovery Mockup (v1)
An interactive mobile-first prototype demonstrating:
-   **Global World Tour**: A horizontal carousel highlighting major international skate events and festivals.
-   **Local Scene Geolocation**: A list-based view for discovering sessions and community events nearby.
-   **Interactive Detail Views**: Full-screen event details with location integration and attendee "stoked" indicators.
-   **Learning Hub**: A filtered gallery of instructional skate videos.
-   **Smart Post Creation**: A simulation of the post creation flow for Pros and Admins.

### 2. Product Roadmap (Overview)
A high-level view of the Let's Roll LABS development focus, including:
-   Event Discovery (Live)
-   Spot Finder (In Progress)
-   Crew Finder (Planned)

## Brand Identity
This project utilizes the official **Let's Roll** brand palette and typography:
-   **Fonts**: 
    -   *Millimetre* (Headings/Display)
    -   *Poppins* (Body)
-   **Colors**: Custom `lr-` tokens (Yellow, Purple, Black, etc.) implemented via Tailwind CSS.

## Deployment
Changes pushed to the `main` branch are automatically deployed via GitHub Pages.
